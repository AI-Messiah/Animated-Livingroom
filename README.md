# Animated-Livingroom
I have made a 3D graphics app to display as many things as I could think of that can be in a living room that can be animated without armatures.

There are also some things that a living room might have that are not animated.

This project is made specifically for Windows but can easily be adapted to Mac or Linux.

I have made an OpenGL version and I might add a Vulkan version later, but I will not include DirectX.

# OpenGL version
requires: GLEW, GLFW, stb_image, Assimp and IrrKlang
